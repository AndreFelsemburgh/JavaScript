Conversão de valores e concatenação

Problema

Perguntar dois números ao usuário e mostrar a soma.

Solução

                // Pedir para o usuário digitar um número
                var n1 = prompt("Digite um número:");
                
                //Converte o valor digitado para número
                n1 = parseFloat(n1);
                
                // Pedir para o usuário digitar outro número
                var n2 = prompt("Digite outro número");
                
                //Converte o valor digitado para número
                n2= parseFloat(n2);
                
                // Mostrar a soma
                alert("A soma entre os números " + n1 + " e " + n2 + " é: " + (n1 + n2));
