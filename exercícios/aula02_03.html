Calcular Média

Problema

Perguntar dois números ao usuário e mostrar a média

Solução

                 // Pedir para o usuário digitar dois números
                var n1 = prompt("Digite um número:");
                
                var n2 = prompt("Digite outro número: ");
        
                // Converter os dados digitados para número
        
                n1 = parseFloat(n1);
                n2 = parseFloat(n2);
        
        
                // Calcular a média
                var media = (n1 + n2) / 2;
                
                // Exibir a média
                alert("A media entre " + n1 + " e " + n2 + " é: " + media);
