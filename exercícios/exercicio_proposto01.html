Operadores aritméticos

Problema

Perguntar dois números ao usuário e mostrar: a subtração, a multiplicação, a divisão e a soma, concatenando com os textos:

  • "A subtração é: ..."
  • "A miltiplicação é: ..."
  • "A divisão é: ..."
  • "A soma é: ..."

Solução

                 // Pedir para o usuário digitar um número
                var n1 = prompt("Digite um número:");
        
                // Pedir para o usuário digitar outro número
                var n2 = prompt("Digite outro número: ");
        
                // Converter os dados digitados para número inteiro
        
                n1 = parseInt(n1);
                n2 = parseInt(n2);
        
        
                // Mostrar a subtração
                alert("A subtração é: " + (n1 - n2));
        
                // Mostrar a multiplicação
                alert("A multiplicação é: " + (n1 * n2));
        
                // Mostrar a divisão
                alert("A divisao é: " + (n1/ n2));
        
                // Mostrar a soma
                alert("A soma é: " + (n1 + n2));
